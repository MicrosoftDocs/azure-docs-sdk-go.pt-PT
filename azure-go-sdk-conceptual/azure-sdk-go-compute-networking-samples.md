# <a name="azure-sdk-for-go-samples-for-compute-and-networking"></a>Amostras do SDK do Azure para Go para computação e rede

A tabela seguinte inclui ligações para amostras selecionadas que demonstram a gestão dos recursos de rede virtual e de computação no SDK do Azure para Go.

Todas as amostras do SDK do Azure para Go estão disponíveis no [GitHub](https://github.com/Azure-Samples/azure-sdk-for-go-samples).

| Nome | Descrição |
|------|-------------|
| [network/network](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/network/network.go) | Crie, atualize, elimine e consulte recursos de rede, incluindo redes virtuais, sub-redes e grupos de segurança de rede. |
| [computação/vm_disk](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/compute/vm_disk.go) | Criar, anexar, desanexar, atualizar e encriptar discos de dados para uma VM. |
| [computação/vm](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/compute/vm.go) | Criar, atualizar, desativar e gerir VMs. |
| [computação/vm_with_availabilityset](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/compute/vm_with_availabilityset.go) | Criar conjuntos de disponibilidade e balanceadores de carga para as VMs. |
| [computação/vm_with_identity](https://github.com/Azure-Samples/azure-sdk-for-go-samples/blob/master/compute/vm_with_identity.go) | Criar e gerir Identidades de Serviço Geridas (MSIs) para as VMs. |
